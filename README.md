## Description
The website is developed using HTML and CSS. It demonstrates the implementation of basic web development concepts including embedding maps, image galleries, contact forms, and responsive design.

## Features Implemented
- Embedded Google Map showing the location of Walchand Institute of Technology, Solapur
- Image gallery displaying college-related images
- Contact form that sends an email notification on form submission
- Responsive layout that works on mobile, tablet, and desktop devices

## Technologies Used
- HTML5
- CSS3

## How to Run
1. Clone or download the repository
2. Open `index.html` in any modern web browser
